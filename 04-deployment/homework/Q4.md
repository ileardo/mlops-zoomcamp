## 1. Initialize new environment

```bash
pipenv --python 3.10.13
```
    

## 2. Install dependencies

```bash
pipenv install pandas pyarrow numpy scikit-learn==1.5.0
```

## 3. Activate pipenv shell

```bash
pipenv shell
```
